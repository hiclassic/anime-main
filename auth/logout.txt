🔸 PHP কোড ব্লকের শুরু।

php
Copy code
session_start();
🔹 session_start()
→ এটি সেশন চালু করে যাতে আমরা সেশন ডেটা access বা destroy করতে পারি।
→ সেশন স্টার্ট না করলে পরের unset() বা destroy() কাজ করবে না।

php
Copy code
session_unset();
🔹 session_unset()
→ এই ফাংশনটি সেশন অ্যারে-এর সব ভ্যারিয়েবল remove করে (value মুছে দেয়)।
→ কিন্তু session ID এখনো সক্রিয় থাকে।

php
Copy code
session_destroy();
🔹 session_destroy()
→ এটি পুরো সেশনটিকে ধ্বংস (destroy) করে দেয়।
→ এটি session ID-সহ সবকিছু সার্ভার থেকে মুছে দেয়।
→ এটি করার পর ইউজার পুরোপুরি লগআউট হয়ে যায়।

php
Copy code
header("location:http://localhost/anime-main/auth/login.php");
🔹 header(...)
→ এটি ইউজারকে redirect করে login.php পেজে।
→ অর্থাৎ logout-এর পর login পেজে পাঠিয়ে দেয়।

✅ এই ফাইলটি যখন চালানো হবে:

সেশন মুছে যাবে (user logout হবে)

তারপর ইউজার login page-এ redirect হবে

🧪 তুমি চেক করতে পারো:
(1) logout.php-তে redirect হচ্ছে কিনা
